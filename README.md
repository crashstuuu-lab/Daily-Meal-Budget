# 每日吃饭预算

这是一个不需要登录、可以直接部署到 Vercel 的极简网页。
数据保存在每台设备自己的浏览器本地。

## 你现在已经有的文件

- `index.html`
- `style.css`
- `app.js`

## 最简单的打开方法

1. 打开这个文件夹：
   `/Users/dangerweakness/Desktop/CODEX小实验/每日吃饭预算`
2. 直接双击 `index.html`
3. 浏览器打开后就可以使用

如果你之后部署到 Vercel，就直接打开部署后的网址。

## 网页怎么用

1. `今天预算`：今天普通预算池里现在有多少钱
2. `今天已花`：今天已经记录了多少钱
3. `剩余金额`：今天预算减掉今天消费后的结果
4. `明天预计预算`：今天花完以后，明天大概能用多少钱
5. `本周剩余额度`：如果这周后面都还没花，这周理论上还剩多少普通预算
6. `设为放纵日`：本周最多可用两次；放纵日当天预算不计入，消费也不会扣正常预算
7. `记一笔`：添加今天消费

## 预算规则

- 每天默认增加 `50` 元普通预算
- 今天超支，明天预算会变少
- 今天节省，明天预算仍然是 `50` 元
- 每周可以有 `2` 天放纵日
- 放纵日当天预算不计入，本周剩余额度也不会计算放纵日那一天
- 放纵日的消费不会扣普通预算
- 数据保存在当前浏览器本地

## 注意

- 你换浏览器、换手机，数据不会自动同步过去
- 如果你清空浏览器本地数据，记录会消失
- 部署到 Vercel 后，手机也能随时访问网页，但数据仍然只存在这台手机自己的浏览器里
